=================
Custom Addons
=================

This repository contains custom Odoo addons.

Modules
=======

Test Module
-----------

A test module for Odoo that demonstrates Python 3.12 features.

Features
^^^^^^^^

* Python 3.12 pattern matching
* Type hints
* Modern Python features

Custom Hello Module
-------------------

A basic hello world module for Odoo that demonstrates basic module structure.

Features
^^^^^^^^

* Basic Odoo module structure
* Hello world functionality

Usage
=====

These modules can be installed like any other Odoo module.
